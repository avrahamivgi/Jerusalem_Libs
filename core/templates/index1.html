Library API Instructions

To use the API, you need to be signed in. If you haven't signed up yet, you can do so here.

This is the base path for the API: http://localhost:8000/api/v1/

Customer API

GET /customers: Retrieve all customers or a specific customer by ID.

POST /customers: Add a new customer.

PUT /customers: Update the name or age of a customer.

DELETE /customers: Delete a customer if they don't have any associated rents.

Book API

GET /books: Retrieve all books or a specific book by name.

POST /books: Add a new book.

PUT /books: Update the return period of a book if it is not currently in rent.

DELETE /books: Delete a book if it doesn't have any associated rents.

Rent API

GET /rents: Retrieve all rents or rents by customer ID. Add the query parameter lated=true to retrieve only late rents.

POST /rents: Add a new rent. Check if the book is already rented.

DELETE /rents: Delete a rent.

User API

POST /signup: Create a new user.

GET /private: Access a private endpoint. Requires authentication.
